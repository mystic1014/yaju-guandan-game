# 雅局掼蛋

一款可离线续玩的单人网页掼蛋游戏：玩家与三名 AI 使用两副牌（108 张）对局，每人 27 张，队友固定坐在正对面。

## 规则基准

规则引擎以 2022 年《竞技掼蛋竞赛规则（试行）》为基准，覆盖：

- 从 2 升至 A、双方独立级牌与过 A 胜负条件
- 逢人配、顺子、三连对、钢板、同花顺、四王炸和多张炸弹
- 接风（界面文案“借风”）、进贡、还贡与抗贡
- 双下升 3 级、头游搭档三游升 2 级、头游搭档末游升 1 级

参考：[国家体育总局规则说明](https://www.sport.gov.cn/n14471/n14481/n14518/c25903605/content.html)。

## 本地运行

要求 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`。

## 测试

```bash
npm test
npm run lint
npx tsc --noEmit
```

`npm test` 会依次运行规则引擎测试、生产构建和服务端渲染验收。浏览器实机验收覆盖 1920×1080、1366×768 与 1024×768 横屏尺寸。

## 存档与隐私

比赛状态、随机种子、贡还阶段和用户设置保存在浏览器本机，不需要账号，不上传牌局数据。存档带版本号；损坏或不兼容的存档会被安全拒绝。
